# Nifty 100 Financial Intelligence Platform

A production-grade financial analytics platform covering 92 Nifty 100 companies.

## Tech Stack
- Python 3.9
- Pandas, NumPy
- SQLite
- Jupyter Notebooks

## Project Structure
nifty100_project/

├── data/

│   ├── raw/          # 7 core Excel datasets

│   └── supporting/   # 5 supplementary datasets

├── src/

│   ├── etl/          # Data loading and cleaning

│   └── analytics/    # Financial ratio engine

├── notebooks/        # Jupyter exploration notebooks

└── output/           # Generated reports and CSVs
